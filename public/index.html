API Reference

Base URL: https://your-domain.com/api — All endpoints return JSON.

GET /api/info Get metadata for a URL
ParamTypeDescription
url requiredstringYouTube or SoundCloud URL
Example Response
{
  "platform": "youtube",
  "id": "dQw4w9WgXcQ",
  "title": "Rick Astley - Never Gonna Give You Up",
  "uploader": "Rick Astley",
  "duration": 212,
  "thumbnail": "https://...",
  "formats": [...]
}
GET /api/search Search by keyword
ParamTypeDescription
q requiredstringSearch query
platformstringyoutube (default) or soundcloud
limitnumberResults count (default: 5)
Example Response
{
  "results": [
    {
      "id": "dQw4w9WgXcQ",
      "title": "Never Gonna Give You Up",
      "uploader": "Rick Astley",
      "duration": 212,
      "thumbnail": "https://...",
      "url": "https://youtube.com/watch?v=dQw4w9WgXcQ",
      "platform": "youtube"
    }
  ]
}
GET /api/download Download file (streams MP3/MP4)
ParamTypeDescription
url requiredstringYouTube or SoundCloud URL
formatstringaudio (MP3) or video (MP4)
qualitystringbest (default) or worst
Returns
Binary file stream with Content-Disposition header
Content-Type: audio/mpeg or video/mp4
GET /api/stream-url Get direct CDN stream URL
ParamTypeDescription
url requiredstringYouTube or SoundCloud URL
formatstringaudio or video
Example Response
{
  "stream_url": "https://rr3---sn-xxx.googlevideo.com/..."
}

🤖 TrashBot Plugin Example

Use the API inside your Baileys bot plugin to let users download music with a simple command like .play Never Gonna Give You Up

// plugins/music.js
const API_BASE = 'https://your-domain.com/api';

export default async (m, { sock, text }) => {
  if (!text) return m.reply('Usage: .play <song name or URL>');

  await m.reply('🔍 Searching...');

  // Search
  const search = await fetch(`${API_BASE}/search?q=${encodeURIComponent(text)}&limit=1`);
  const { results } = await search.json();
  if (!results.length) return m.reply('❌ No results found.');

  const song = results[0];
  await m.reply(`🎵 Found: *${song.title}*\nDownloading...`);

  // Get download URL
  const dl = await fetch(`${API_BASE}/download?url=${encodeURIComponent(song.url)}&format=audio`);
  const buffer = Buffer.from(await dl.arrayBuffer());

  await sock.sendMessage(m.chat, {
    audio: buffer,
    mimetype: 'audio/mpeg',
    fileName: `${song.title}.mp3`,
    ptt: false,
  }, { quoted: m });
};
